📌 Pineapple Rush - Um Jogo Desenvolvido com Construct 3 🎮 Sobre o Projeto O Pineapple Rush é um jogo criado como parte de um projeto acadêmico no Centro Universitário Integrado. Este projeto visa explorar a criação de jogos utilizando abordagens low-code, permitindo que desenvolvedores consigam construir experiências interativas sem necessidade de programação avançada.

🛠️ Desenvolvimento com Low-Code A criação de Pineapple Rush foi realizada através do Construct 3, uma poderosa ferramenta de desenvolvimento de jogos voltada para low-code. A plataforma permite que jogos sejam desenvolvidos com uma interface visual intuitiva e lógica baseada em eventos, reduzindo a complexidade de programação tradicional.

Com Construct 3, foi possível:

Criar mecânicas interativas sem código manual extenso

Implementar animações, física e lógica do jogo de forma visual

Exportar o jogo diretamente para HTML5, permitindo que ele seja jogado no navegador

🌟 A Experiência de Criar um Jogo Criar um jogo do zero é uma experiência incrível e enriquecedora. Desde o conceito inicial, passando pela construção das mecânicas e ajustes gráficos, Pineapple Rush representa aprendizado prático e criatividade aplicada no desenvolvimento de jogos digitais.

Desenvolver com low-code expande possibilidades para novos criadores, permitindo que ideias inovadoras saiam do papel sem barreiras técnicas, ao mesmo tempo que introduz conceitos fundamentais de game design, lógica de programação e experiência do usuário.
